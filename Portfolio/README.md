## Installation & Setup  
1. Clone the repository:  
   ```sh
   git clone https://github.com/Abhi-upadhyay2109/portfolio-website.git
   cd portfolio-website
   ```  
2. Install dependencies:  
   ```sh
   npm install
   ```  
3. Start the development server:  
   ```sh
   npm run dev
   ```
